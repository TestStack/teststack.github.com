TestStack is a collection of awesome open source testing related projects which mainly focus on making the left side of Marick’s agile testing quadrant (introduced in the book Agile Testing) easier!

TestStack is focused the left side of the quadrant (Q1 and Q2) which are tests to support the agile team of developers, testers and customer. They are test first, promoting the “testing as design” mindset, and tend to be automated. Things like unit and integration tests, acceptance tests, and UI tests and methodologies like Behaviour Driven Development and Acceptance Test Driven Development.

But in short, TestStack is just a bunch of awesome testing related projects!

TestStack.BDDfy

BDDfy (pronounced B D Defy) is the simplest BDD framework for .Net EVER while also being incredibly powerful! The name comes from the fact that it allows you to turn your tests into BDD behaviors simply.

It supports:

  • Running in any test framework (nUnit, xUnit, MSTest etc)
  • Writing your BDD tests in code, with type safety
  • Reporting your tests using the Gherkin DSL in many formats like Html, text, markdown and is fully extensible!
  • Full support for cucumber examples, either text tables OR in a type safe way
  • And much, much more!
